Detailed Description

Gadget class.

Library: ilvadvgdt

The IlvText class defines a multiline text editor that can contain up to 65535 lines of text. The contents of a text gadget can be edited.

A Multiline Text Gadget

See also
IlvTextLocation, IlvTextField.

Accessors

Accessors provide a scriptable and uniform way to inspect and modify an object by using its base class methods IlvValueInterface::queryValue(), IlvValueInterface::queryValues(), IlvValueInterface::changeValue(), IlvValueInterface::changeValues(). This class inherits the accessors of its superclass IlvScrolledGadget and adds the following ones:

text String getText(), setText()

editable Boolean isEditable(), setEditable()
cursorLocation UInt getCursorLocation(), setCursorLocation()
selectionStart UInt getSelection(), setSelection()
