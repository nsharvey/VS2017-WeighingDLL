Detailed Description

View rectangle class.

Library: ilvgadgt

The IlvSCGadgetContainerRectangle class implements an IlvGadgetContainerRectangle with scroll bars. You can decide whether the scroll bars should be shown or hidden. This class creates an IlvScrolledView object containing the gadget container encapsulated by this object.

See also
IlvScrolledView
