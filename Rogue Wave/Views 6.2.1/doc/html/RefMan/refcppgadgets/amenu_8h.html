§ IlvMenuItemType

enum IlvMenuItemType

This enumeration type describes the different types of menu items. Only IlvSeparatorItem and IlvTearOffItem are meaningful. Other values have been kept for compatibility.

Enumerator
IlvSeparatorItem 

The item is a separator.

IlvTearOffItem 

The item is a tear-off item.
