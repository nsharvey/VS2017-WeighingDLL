Detailed Description

Gadget class.

Library: ilvadvgdt

The IlvScrolledGadget class is an abstract gadget class that handles scroll bars. It is the base class for the gadgets that have scroll bars.

Examples of subclasses of the IlvScrolledGadget class

See also
IlvAbstractMatrix, IlvAnnoText, IlvScrollBar, IlvStringList, IlvText, IlvTreeGadget.

Accessors

Accessors provide a scriptable and uniform way to inspect and modify an object by using its base class methods IlvValueInterface::queryValue(), IlvValueInterface::queryValues(), IlvValueInterface::changeValue(), IlvValueInterface::changeValues(). This class inherits the accessors of its superclass IlvGadget and adds the following ones:

horizontalScrollbar Boolean getScrollBar(), showScrollBar(), hideScrollBar()
verticalScrollbar Boolean getScrollBar(), showScrollBar(), hideScrollBar()

verticalScrollbarAsNeeded Boolean getScrollBarShowAsNeeded(), scrollBarShowAsNeeded()

verticalScrollbarPosition Direction whereIsScrollBar(), moveScrollBar()
