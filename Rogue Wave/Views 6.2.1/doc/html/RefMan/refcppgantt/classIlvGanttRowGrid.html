Detailed Description

Grid for Gantt rows in an IlvGanttChart.

Library: ilvgantt

This class draws horizontal grids for the Gantt rows in the grapher views of the IlvGanttChart.

See also
IlvGanttChart, IlvGanttHorizontalGrid.
