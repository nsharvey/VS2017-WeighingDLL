Detailed Description

Abstract class that represents an area of a logical page.

Library: ilvprint

This class is used to represent one of the different areas of the logical page. These areas, in fact rectangles, are:

  • background (usually used to print on all the page in the background);
  • header (usually used to print at the top of the page);
  • main (usually used to print the main area of the page);
  • footer (usually used to print at the bottom of the page);
  • foreground (usually used to print on all the page in the foreground).
