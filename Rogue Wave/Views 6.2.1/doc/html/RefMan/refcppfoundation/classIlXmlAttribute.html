Detailed Description

Default implementation of XML attributes.

Library: ilog

This class defines a default implementation of XML attribute objects. It is provided for convenience, so that you can manipulate XML attributes without having to create a subclass from IlXmlAttributeI. This class actually stores the key-pair values as strings.
