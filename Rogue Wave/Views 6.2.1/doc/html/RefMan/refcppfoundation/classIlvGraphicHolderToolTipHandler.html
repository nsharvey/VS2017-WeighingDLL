Detailed Description

Tooltip handler class.

Library: views

The class IlvGraphicHolderToolTipHandler manages tooltips in a view connected to an IlvGraphicHolder object (an IlvContainer, or an IlvView connected to an IlvManager).

See also
c IlvToolTip, IlvView, IlvGraphicHolder
