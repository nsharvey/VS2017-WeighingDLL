Detailed Description

Netscape fixed colormap quantizer class.

Library: xviews or winviews or mviews (mutually exclusive)

This class provides methods to convert true color images to indexed images using a fixed colormap. (The colormap is not computed from the source image). The colormap is defined by Netscape and uses 216 colors.
