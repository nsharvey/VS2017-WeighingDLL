Detailed Description

This class is used to temporarily modify the clip of an IlvPalette.

Library: xviews or winviews or mviews (mutually exclusive)
If the pointer to the new clip is not 0, the constructor modifies the clip of the palette with the intersection of the current clip and the new one, while the destructor restores the previous value. Otherwise the clip remains unchanged.
