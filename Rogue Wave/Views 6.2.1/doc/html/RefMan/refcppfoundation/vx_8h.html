#define ILVVXLOOK
 

Detailed Description

Library: xviews or winviews or mviews (mutually exclusive)
Definition of the ILVVXLOOK symbol. This file must be included before any other include file to register the ViewsX look and feel. You need to include this header file only when using static libraries. In this case, you must link your application with the ViewsX look libraries. See the information on "look-and-feel" in the Rogue Wave Views Controls User's Manual.
