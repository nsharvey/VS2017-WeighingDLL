Detailed Description

Manager command class.

Library: ilvmgr

The class IlvDeleteObjectsCommand is a subclass of IlvManagerCommand that must be instantiated once several objects have been removed from an IlvManager to record the modification and make it possible to undo or redo it.
This class is provided for compatibility purposes only.
