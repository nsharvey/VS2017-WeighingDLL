Detailed Description

Handles interactions on IlvSCManagerRectangle instances.

Library: ilvgadmgr

This interactor must be used to enable scrolling interactions with IlvSCManagerRectangle instances. The shared instance of the IlvSCManagerInteractor class is automatically set to a IlvSCManagerRectangle object when it is added to an IlvContainer or an IlvManager.

The registered name of this interactor is "SCManager".
