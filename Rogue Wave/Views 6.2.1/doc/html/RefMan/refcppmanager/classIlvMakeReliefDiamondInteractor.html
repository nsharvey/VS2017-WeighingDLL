Detailed Description

Manager interactor class.

Library: ilvmgr

Use the global function IlvGetSymbol to get or create a symbol.

See also
IlvManager, IlvRectangle, IlvMakeRectangleInteractor, IlvMakeReliefRectangleInteractor, IlvMakeRoundRectangleInteractor, IlvMakeEllipseInteractor, IlvMakeBitmapInteractor.
