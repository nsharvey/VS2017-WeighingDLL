§ IlvDoubleFunction

IlvDoubleFunction

Defines the type of the callback function used within a IlvCallbackChartFunction object.

Defines the type of the callback function used to define the data points represented by a data set of type IlvCallbackChartFunction. The callback function computes the ordinate value of a data point from the abscissa value that is passed as a parameter.
