Macro Definition Documentation

§ IlvChartDisplayerDepth

#define IlvChartDisplayerDepth

This constant defines the default depth used for the displayers. For example, it is used to define the default depth of the 3D bars for a 3D bar displayer.
The value of this constant is 4.

§ IlvChartDisplayerWidth

#define IlvChartDisplayerWidth

This constant defines the default width used for the displayers. For example, it is used to define the default width of the bars for a bar displayer.
The value of this constant is 8.
