Detailed Description

This class is an IlvMapTileLoader specialized for loading a set of images, each image representing a tile in a tiled data set.

Library: ilvmaps

Constructor & Destructor Documentation

§ IlvImageTileLoader()

IlvImageTileLoader::IlvImageTileLoader ( IlvDisplay *  display,
const char *  pattern,
const char *  colFormat,
const char *  rowFormat,
const IlvMapAdapter *  adapter 
)

constructor.

Creates an IlvImageTileLoader by specifying a pattern and two formats used to construct a file name for a given IlvTile. The pattern string must contain one r and one c specifier. When a tile is to be loaded, a file name is built by replacing the r conversion specifier with the row index of the tile, using the row format string. The same operation is performed for the column of the tile, replacing the c conversion specifier with the column index of the tile, using the column format string. For example, if the pattern is "<tt>ADirectory/file_%c_%r_image.jpg</tt>" and the colFormat is "%03d" and the colFormat is "%04d", the resulting file name for the tile at (12, 34) will be "<tt>ADirectory/file_0034_012_image.jpg</tt>".

Parameters
displayThe display.
patternThe file name pattern, containing one 'c' and one 'r'.
colFormatA formatting string that is interpreted as the regular C function printf used as the column specifier.
rowFormatA formatting string that is interpreted as the regular C function printf used as the row specifier.
adapterThe IlvMapAdapter.
