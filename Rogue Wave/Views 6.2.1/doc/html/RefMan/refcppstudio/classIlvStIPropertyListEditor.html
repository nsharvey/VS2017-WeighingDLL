Detailed Description

Inspector class.

Library: ivstudio

This class defines a list editor. It handles a number of gadgets that are used when editing a list. These gadgets are the following:

  • A list gadget, showing the list that is being edited.
  • A button that adds an item after the current selection when pressed.
  • A button that inserts an item before the current selection when pressed.
  • A button that removes the current selection when pressed.
  • A button that cleans the edited list.

The use of these various gadgets causes the methods of the class IlvStIPropertyListAccessor to be called.

See also
IlvStIPropertyListAccessor.
