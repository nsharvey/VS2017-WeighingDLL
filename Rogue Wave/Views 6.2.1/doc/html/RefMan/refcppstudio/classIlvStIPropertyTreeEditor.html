Detailed Description

Inspector class.

Library: ivstudio

This class defines a tree editor. It handles a number of gadgets that are used when editing a tree. These gadgets are the following:

  • A tree gadget, showing the tree that is being edited.
  • A button that adds an item after the current selection when pressed.
  • A button that inserts an item before the current selection when pressed.
  • A button that adds a child tree item to the current selection when pressed.
  • A button that removes the current selection when pressed.
  • A button that cleans the edited tree.

The use of these various gadgets causes the methods of the class IlvStIPropertyTreeAccessor to be called.

See also
IlvStIPropertyTreeAccessor.
