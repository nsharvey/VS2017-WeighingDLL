Detailed Description

Gadget class.

Library: dbsqlgad

The IliSQLTablesUsesGadget class defines a tree gadget class that can display up to three levels of information for each valid data source of the application. The levels are:

  • The SQL table name
  • The data source name of each table
  • The column name of each table for each data source

Each data source has a bitmap showing if the data source is global. If the data source is global the bitmap is green. If the data source is not global, the bitmap is red.
