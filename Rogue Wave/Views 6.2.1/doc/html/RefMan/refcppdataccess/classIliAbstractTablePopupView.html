Detailed Description

Dialog class.

Library: dbgadget

This class is an abstract class that defines the dialog used by the combo box displaying the pull-down menu. To associate a combo box with a dialog, you should use the IliTableComboBox::setUserComboBoxWindow() member function.

See also
IliTableComboBox, IliTablePopupView.
