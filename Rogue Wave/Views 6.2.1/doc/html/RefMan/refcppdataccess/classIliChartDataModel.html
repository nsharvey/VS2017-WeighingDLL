Detailed Description

A model class.

Library: dbchart

The class defines a chart data model. A chart data model is used by the IliChartGraphic class to handle the data of one or several data sources. All chart data models inherit of this class. There are three predefined data model:

  • By row: Each row corresponds to a point for a series. This model uses a data source with three columns:
    • Series, at position 0 when you use the column functions of the chart.
    • x axis, at position 1 when you use the column functions of the chart.
    • value, at position 2 when you use the column functions of the chart.
  • By Column: Each row corresponds to a point for each series for one x-axis value. This model uses a data source with one column for x-axis and one column for each series:
    • x axis, at position 0 when you use the column functions of the chart.
    • series 1, at position 1 when you use the column functions of the chart.
    • series n, at position n when you use the column functions of the chart.
  • By data source: For each series there is a data source which contains all points of the series. Each data source has got two columns:
  • x axis, at position 0 when you use the column functions of the chart.
  • value, at position 1 when you use the column functions of the chart.
See also
IliChartGraphic, IliChartSeriesModel, IliModel, IliGadgetModel.
