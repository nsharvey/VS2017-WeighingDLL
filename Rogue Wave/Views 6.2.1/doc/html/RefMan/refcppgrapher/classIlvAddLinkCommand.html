Detailed Description

Grapher command class.

Library: ilvgrapher

The class IlvAddLinkCommand is a subclass of IlvManagerCommand. IlvAddLinkCommand must be instantiated once a link object has been added to an IlvGrapher. The IlvGrapher class inherits from IlvManager to record the modification and make it possible to undo or redo it.
This class is provided for compatibility purposes only.
