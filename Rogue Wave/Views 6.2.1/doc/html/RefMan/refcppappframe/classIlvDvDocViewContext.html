Detailed Description

The IlvDvDocViewContext class is used internally by the IlvDvDocViewInterface class.

Library: ilvappframe

An IlvDvDocViewContext object is created and handled by an IlvDvDocViewInterface object to store all its internal data, such as the attached document and the document view container.
