The IlvShaping Attribute
This Boolean attribute determines whether characters should be shaped before they are displayed. This attribute is relevant only if characters can have different shapes according to their position in a word. This is the case of Arabic, Farsi, and Urdu. Hebrew is not considered a contextual language.
Possible Values
This attribute can have one of the following values:
IlvShapingOff - Shaping is not performed.
IlvShapingOn - Shaping is performed.
Limitation
In Rogue Wave Views Bidi for Windows, this attribute is not supported; only the value IlvShapingOn is.
