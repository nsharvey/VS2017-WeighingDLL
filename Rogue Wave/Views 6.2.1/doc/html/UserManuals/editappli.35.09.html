The Script Page
The Script page can be used to specify the use of Rogue Wave Script.
Script Page in the Application Inspector
The Script page contains the following fields:
Use Rogue Wave Script  Select this toggle button if you want to use Rogue Wave Script.
Auxiliary Library  Select this toggle button if you want to use the auxiliary library of Rogue Wave Script for Rogue Wave Views in your application through the scripting language. This library lets you use additional features, such as the dialog boxes. For more information, see the chapter “Rogue Wave Script Programming” in the Rogue Wave Views Foundation User’s Manual.
Script File  Enter the name of a file containing script code or click the button next to the text field to display a File Chooser to select a file.
