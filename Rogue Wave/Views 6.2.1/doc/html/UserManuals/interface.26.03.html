The Menu Bar
The menu bar in the Main window provides you with general Rogue Wave® Views Studio commands. When you highlight a command item, the Message area gives you a short description of the command. The following tables summarize the commands you can execute through the menu bar. For details on these commands, see Commands, where they are listed in alphabetical order.
