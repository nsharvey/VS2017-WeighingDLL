Studio > Commands > Predefined Commands > SameWidth
 
SameWidth
Label
Equal Width
Path
Main window: Draw menu > Align/Distribute; Alignment panel
Category
align
Action
Resizes the selected objects so they have the same width. Object positions are not changed. If only one object is selected, it is resized so it has the same width as the panel or the distance between the two guides to which it may be attached. If at least two objects are selected, the first selected object is used as the reference.
