Loading and saving Graphic Objects
Rogue Wave Views can load and save data files that contain the definition of graphic object instances that must be persisted. Although this section describes how to deal with persistence of the IlvGraphic class and its sub-classes, there are several base classes that expose the serialization and deserialization functionality in a similar manner.
This section covers the following topics:
Persistence basics
Accessing the deserialization constructor
Dynamic loading
