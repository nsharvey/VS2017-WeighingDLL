The IlvSwapping Attribute
Certain nonalphabetic characters, such as parentheses, have a visual orientation such that they must be replaced with a symmetrical character when they are in a right to left context. The parameter of type IlvSwapping determines whether these characters should be replaced.
Possible Values
This attribute can have one of the following values:
IlvSwappingOff - The characters should not be replaced.
IlvSwappingOn - The characters should be replaced.
