Menus Palette
The Menus palette contains the following objects that can be created using the normal creation mode of Rogue Wave® Views Studio or the drag-and-drop operation.
Type
Icon
Description
Menu Bar
Creates an IlvMenuBar object.
Pop-up Menu
Creates an IlvPopupMenu object. If you want to attach a pop-up menu to a menu bar or another pop-up menu, you must use the Menu editing mode. Click the Menu icon in the Editing Modes toolbar.
Toolbar
Creates an IlvToolBar object.
