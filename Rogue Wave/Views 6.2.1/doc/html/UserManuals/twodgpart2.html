Grapher
Part II describes a high-level Rogue Wave® Views package called the grapher, which is dedicated to the graphic representation of hierarchical and interconnected information. This part consists of the following chapters:
Introducing the Grapher Extension of Rogue Wave Views Studio describes how to use the Grapher extension of Rogue Wave Views Studio.
Features of the Grapher Package describes the classes, methods, and principles that make the Grapher package work.
Note: The Rogue Wave Views Grapher package is available only if you have purchased the Rogue Wave Views 2D Graphics Professional product.
