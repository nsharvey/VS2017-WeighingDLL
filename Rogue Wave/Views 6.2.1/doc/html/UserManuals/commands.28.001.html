Commands
A command is a Rogue Wave® Views Studio basic entry point. It is mainly defined by an action, a label, a prompt message, and often an icon. It can be attached to menu and toolbar items, buttons, and other active gadgets in any Rogue Wave Views Studio panel.
This chapter introduces general command definitions and describes the Rogue Wave Views Studio predefined commands.
