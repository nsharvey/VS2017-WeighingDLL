Limitations
The following table summarizes the limitations and unsupported features of the use of Cairo through Rogue Wave Views:
Cairo Limitations with Rogue Wave Views
Rendering
Cairo rendering in Rogue Wave Views only applies on bitmaps. In most use cases, setting the double-buffering on in your application is a workaround for this limitation.
Printing
Cairo rendering is not supported in Rogue Wave Views.
Drawing text
You cannot draw text using Cairo in Rogue Wave Views, so you will not able to get alpha-blended or antialiased text drawings on X Window systems using Rogue Wave Views.
Draw Mode
Cairo does not support any other mode than IlvModeSet.
