AddZoomableIcon
Deprecated - use creation mode instead.
Label
IlvZoomableIcon
Category
add
Action
Lets you insert an IlvZoomableIcon object in the work space. This command opens a bitmap File Selector panel that lets you choose a bitmap. After choosing a bitmap, click in the work space to indicate the position of the object.
