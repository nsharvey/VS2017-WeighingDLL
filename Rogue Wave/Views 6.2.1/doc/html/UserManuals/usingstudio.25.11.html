Printing
To do a printing you can select one of the following items from the File menu:
Print the current buffer
Preview the current buffer
Print...
Preview...
The first two items, Print the current buffer and Preview the current buffer, allow you to print or preview the current buffer. The layout of the current buffer is defined by means of the IlvPrintableLayoutIdentity class. The header associated with this layout contains the file name, while the footer contains the index of the page/the total number of pages. You cannot modify this layout.
The last two items, Print and Preview, allow you to print or preview a list of buffers or texts. You can also choose the layout, and several other properties.
