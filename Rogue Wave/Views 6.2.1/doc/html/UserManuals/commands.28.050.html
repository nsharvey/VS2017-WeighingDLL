OpenDocument
Label
Open…
Path
Main window: File menu and Open icon in the toolbar
Category
document
Action
Loads all the file types that are declared by the fileLoader option. This option is documented in Customizing Rogue Wave Views Studio. The following are the predefined file extensions:
ilv executes OpenBuffer
iva executes OpenApplication
dbm executes LoadMessageDatabase
js executes LoadScript (jsstudio only)
