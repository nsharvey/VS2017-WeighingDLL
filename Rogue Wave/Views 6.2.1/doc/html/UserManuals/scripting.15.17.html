Using Resource Names with Rogue Wave Script for Rogue Wave Views
To modify the resource associated with a given Rogue Wave Views object (whether a color, a pattern, a line or fill style, an arc mode, and so on) use its name. Here are a few examples:
myButton.foreground = "red";
myButton.pattern = "solid";
myLabel.alignment = "right";
Resource names are listed in Resource Names.
