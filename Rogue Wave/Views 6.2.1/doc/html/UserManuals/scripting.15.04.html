Binding Rogue Wave Views Objects
To use Rogue Wave® Views objects, such as gadgets and panels, in Rogue Wave Script, you must make these objects accessible using a binding procedure. To bind an object:
First get a Rogue Wave Script context. This is described in Getting the Global Rogue Wave Script Context.
Invoke the bind method, discussed in Binding Rogue Wave Views Objects.
A bound object becomes accessible from Rogue Wave Script.
