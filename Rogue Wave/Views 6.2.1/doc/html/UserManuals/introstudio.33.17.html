Generate
Label
Generate
Path
Main window: Application menu
Category
application
Action
Saves the application description file, and generates the C++ code for the application and its modified panel classes. If the application file name is the default name, a File Selector panel is opened to let you enter a new file name. If the Make toggle button in the Application Inspector is turned on, the make file is also generated.
