This manual contains three separate parts divided into chapters, and one Appendix. Each of these parts describes one of the packages that make up Rogue Wave® Views Controls.
Part I, Creating GUI Applications with Rogue Wave Views Studio describes how to use Rogue Wave Views Studio with the Gadgets extension installed.
Part II, Rogue Wave Views Gadgets provides information for developing applications that incorporate Rogue Wave Views Gadgets.
Part III, Rogue Wave Views Application Framework describes how to use the Application Framework package of Rogue Wave Views.
Editing States provides an example of how to use the state mechanism of Rogue Wave Views Studio.
