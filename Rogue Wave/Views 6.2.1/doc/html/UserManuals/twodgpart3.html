Part III 
Prototypes
Part III explains the concepts behind Business Graphic Objects and shows how to create and use prototypes with both containers and managers. This part consists of the following chapters:
Introducing the Prototypes Package introduces the concepts of prototypes.
The User Interface and Commands describes the main parts of Rogue Wave® Views Studio with the Prototypes extension.
Using Rogue Wave Views Studio to Create BGOs explains how to use Rogue Wave Views Studio to create your prototypes by composing graphic objects and assigning behaviors to them.
Using Prototypes in C++ Applications describes the classes and methods used to manipulate prototypes and shows how to structure your application to benefit fully from BGOs. It then shows how to use prototypes created in Rogue Wave Views Studio in your C++ applications.
Predefined Accessors lists the behaviors that are predefined in the Prototypes library.
Note: The Rogue Wave Views Prototype package is available only if you have purchased the Rogue Wave Views 2D Graphics Professional product.
