Studio
This manual contains the following chapters:
Introducing Rogue Wave Views Studio provides set-up information, explains how to launch Rogue Wave® Views Studio, and briefly presents the panels that appear at start-up.
Using Rogue Wave Views Studio helps you get acquainted with the basic components of Rogue Wave Views Studio and explains how to perform the basic tasks you will need to create graphic objects. You will learn how to edit these objects using various tools and commands available in Rogue Wave Views Studio.
The Rogue Wave Views Studio Interface provides a detailed description of the Rogue Wave Views Studio interface.
Using Rogue Wave Script introduces you to an extension of Rogue Wave Views Studio designed to facilitate the writing and editing of Rogue Wave Script-based callbacks that can be attached to Rogue Wave Views objects.
Commands provides a description of the commands available in Rogue Wave Views Studio.
Panels describes the various panels and dialog boxes that you will encounter when using Rogue Wave Views Studio.
Customizing Rogue Wave Views Studio provides information on how to customize Rogue Wave Views Studio using configuration files.
