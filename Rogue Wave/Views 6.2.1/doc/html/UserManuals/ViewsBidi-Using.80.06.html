The IlvNumerals Attribute
Numerals can be written with the Latin alphabet (Arabic numerals) or with the local alphabet (Hindi numerals).
The logical string always stores numerals in the same way, regardless of the selected display mode. The visual shape of numerals is determined by a parameter of type IlvNumerals.
Possible Values
This attribute can have one of the following values:
IlvNumeralsNominal - Numerals are written with the Latin alphabet.
IlvNumeralsNational - Numerals are written with the Hindi alphabet.
IlvNumeralsContextual - The shape of numerals depends on their context. These are written with the Latin alphabet if they belong to a Latin substring, and with the Hindi alphabet if they appear in an Arabic substring. In this mode, it is the system that defines the criteria determining the shape of numerals.
