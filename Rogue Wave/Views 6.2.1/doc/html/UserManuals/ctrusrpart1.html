Part I 
Creating GUI Applications with Rogue Wave Views Studio
This part describes how to use Rogue Wave® Views Studio with the Gadgets extension installed. It contains the following chapters:
Introducing the Gadgets Extension of Rogue Wave Views Studio provides basic information for using the Gadgets extension of Studio. It explains how to launch the GUI applications plug-in, describes the additional interface items that differ from the Foundation Studio, contains a list of additional Studio commands, and describes the interface panels of the Gadgets extension.
Editing Gadget Panels tells you how to create a panel by adding gadgets to it and how to edit these objects using the various tools and commands available in Rogue Wave Views Studio.
Editing Applications tells you how to create applications using Rogue Wave Views Studio.
Using the Generated Code explains how to create a miniature application made up of two panels and how to generate the C++ code for that application.
Customizing the Gadgets Extension of Rogue Wave Views Studio provides a list of configuration options that you can use to customize Rogue Wave Views Studio when the Gadgets extension is installed.
Extending Rogue Wave Views Studio tells you how to adapt Rogue Wave Views Studio to your specific requirements.
Using Inspector Classes tells how to define a new inspector panel and incorporate it into Rogue Wave Views Studio.
At the end of this manual, you will also find the following appendix:
Editing States shows how to use Rogue Wave Views Studio to set up different states for application panels.
