Disk Space and Memory
Disk space and memory requirements for running Rogue Wave® Views and Rogue Wave Views applications are as follows:
Disk space  Disk space requirements vary depending on the target platform and the components installed.
A full installation of one single platform, with all options, requires from 600 MB to 1 GB of free disk space.
Memory  The recommended configuration for running Rogue Wave Views applications is 2 GB or more on 32-bit platforms and 4 GB or more on 64-bit platforms.
