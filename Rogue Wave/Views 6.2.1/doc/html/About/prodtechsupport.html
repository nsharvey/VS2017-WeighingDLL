Rogue Wave Software Support
If you have problems installing, unlocking, or running your software, contact Rogue Wave Software Technical Support. Contact information is found at www.roguewave.com/support/contact-support.aspx.
Please be prepared to provide the following information when you call for consultation during Rogue Wave Software’s business hours:
Name and version number of the product. For example, Rogue Wave Views version 5.5.
The type of system on which the software is being run. For example, UltraSPARC, Intel x86, and so forth.
The operating system and version number. For example, HP-UX 11.0 or Windows XP.
The license number.
A detailed description of the problem.
