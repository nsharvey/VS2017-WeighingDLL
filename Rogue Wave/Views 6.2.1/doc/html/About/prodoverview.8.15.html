Rogue Wave Views Studio
Rogue Wave® Views Studio is a powerful editor for creating, editing, and saving Rogue Wave Views objects. With Rogue Wave Views Studio you can design a panel, edit gadgets, draw and import graphic objects, group objects for editing, assign attributes and multilingual labels to objects, and, in general, create your GUI.
This executable is designed to be supplemented by dynamic modules for each specific use (Studio for Gadgets, Studio for Manager, and so on).
Main Studio Features
The main features of Rogue Wave Views Studio include:
WYSIWYG editor including drag-and-drop and testing mode.
Drag-and-drop.
Marking menus: Rogue Wave Views Studio provides a new kind of pop-up menu: the Marking Menus. These are directional pop-up menus that let you choose one or several items by moving the mouse in a certain direction. They are very easy to use, and very efficient when you are used to them.
Rogue Wave Views Studio Marking Menus
MDI (Multiple Dockable Interface), dockable panels and toolbars.
Platform look and feel: With Rogue Wave Views Studio you can specify the platform look-and-feel with a simple menu selection.
Rogue Wave Views Studio Look-and-feel on Motif® platform
Rogue Wave Views Studio Look-and-feel on Windows® XP platform
Access to Rogue Wave Script.
C++ code generation when relevant.
Customization files.
Extension by plug-ins. You can supplement Rogue Wave Views Studio by adding plug-ins. Most Rogue Wave Views packages provide plug-ins that bring new graphic objects and their inspectors, data or code generation, drawing tools, and other extensions.
Recompilable. In addition to plug-ins, Rogue Wave Views Studio provides libraries to link with your code. Rogue Wave Views Studio can then become your own application editor. This functionality requires that you have access to the Gadgets, Manager, and Grapher packages.
