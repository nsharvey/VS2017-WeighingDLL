The ViewFile Application: Building a Simple File Browser
This tutorial shows you how to create the ViewFile application using the Rogue Wave® Views Gadgets API. The ViewFile application is a small graphical user interface for browsing through a list of files stored on a hard disk.
Note: You can also build the same kind of application using Rogue Wave Views Studio, the Rogue Wave Views GUI editor.
As you progress through this tutorial, you will learn how to combine many different Rogue Wave Views gadgets, such as the tree gadget, the sheet, and the toolbar to produce very graphical and highly intuitive applications. Your final application will include sophisticated features such as docking toolbars and multiple frames.
This tutorial has four steps:
Step 1: Building the Browser Window
Step 2: Browsing Files
Step 3: Adding Docking Bars
Step 4: Adding View Frames
