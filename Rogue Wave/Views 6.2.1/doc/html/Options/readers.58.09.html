Graphical Rendering of a Digital Terrain Model
The default renderer of an IlvDTEDReader class is an IlvDefaultRasterRenderer that displays the raster as an image, with each color corresponding to an elevation value. The default parameters for this renderer use a color model specially designed for displaying elevations (MakeElevationColorModel()). Negative elevations appear in a deep blue, while terrain elevations that are null are rendered with a clear blue. Positive elevations up to one meter are represented in yellow. Higher elevations are represented first in green, then in brown, and finally in white at the highest altitudes.
