Projecting Out-of-Bounds Data Points
By default, the data points that should appear outside of the data display area (if such data points exist) are not drawn. The graphical representations of such data points are simply clipped by the data display area. However, for the displayers that are instances of the IlvPolylineChartDisplayer class or one of its subclasses, it is possible to display the out-of-bounds data points projected on the limits of the data display area. For details, see Polyline Displayer.
