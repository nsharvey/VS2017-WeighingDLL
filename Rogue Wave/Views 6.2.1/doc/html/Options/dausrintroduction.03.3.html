Supported Databases
You can use Data Access with the following databases:
Oracle
Informix
Sybase
OLE DB (only for Windows)
ODBC (only for Windows)
Note: Fetching data from a time column may fail for ODBC when connected to a SQL Server. This is because some versions of the SQL Server driver return SQL_SS_TIME2 as a mapping to the Time data type, which is a non-standard ODBC SQL data type. Non-standard ODBC SQL data types are not supported by the ODBC driver.
DB2
Note: Check the compatibility of your particular database version directly with Rogue Wave support.
