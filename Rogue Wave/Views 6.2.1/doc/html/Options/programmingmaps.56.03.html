Map Features
When you read a map in Rogue Wave® Views, cartographic data is loaded as IlvMapFeature objects. A map feature is an object that represents cartographic data as it was read from its source file. It can be a segment of road, an aerial image, the summit of a hill, or a digital terrain model. A map feature carries the following information:
The geometry of the feature
Attributes
The projection in which the geometry is expressed
Projections are not covered in this section. For details, see Selecting a Target Projection and Map Projections.
