Charts
This manual is divided into two parts that describe how to use the Rogue Wave® Views Charts Package, and one Appendix.
Part I, Using Charts with Rogue Wave Views Studio provides an introduction to the Rogue Wave Views Charts package by presenting several examples of the types of charts and the customizing that can be performed with this package. You will learn how to use the Charts extension of Rogue Wave Views Studio, which is dedicated to the Rogue Wave Views Charts package. It contains the following chapters:
Introducing Charts in Rogue Wave Views Studio
Customizing Charts
Using Polar Charts
Part II, Using the Charts Library gives information on the Charts Library of Rogue Wave Views. It contains the following chapters:
Introducing the Charts Library
Chart Basics
Data Handling
Chart Layout
Data Display
Scales Display
Decorations Display
Interacting with Charts
Using Charts to Display Real-Time Data
The IlvXMLChartData Class
