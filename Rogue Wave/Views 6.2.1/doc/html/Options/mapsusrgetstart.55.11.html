Saving the File
At this point, you are going to save your file. Rogue Wave® Views Map Builder saves files exclusively in the .ilv format. Although this is automatic, add the .ilv extension to the name of the file so that it can be easily identified as a file of this format.
To save the file:
1. Choose Save as... from the File menu.
2. Select the directory you want to save the file in.
3. Name the file Philippines.ilv.
4. Click Save.
