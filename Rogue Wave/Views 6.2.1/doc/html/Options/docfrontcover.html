Maps
This manual provides conceptual and hands-on information for developing applications that incorporate Rogue Wave® Views Maps. It describes the fundamentals that underlie Rogue Wave Views Maps and shows how to create and use map objects.
This manual contains the following chapters:
Introducing Rogue Wave Views Maps gives you an overall picture of Rogue Wave Views Maps.
Getting Started with Rogue Wave Views Maps provides a simple tutorial on the mapping functionalities of Rogue Wave Views Maps.
Rogue Wave Views Maps Reader Framework describes the API to load cartographic data into Rogue Wave Views.
Using Load-On-Demand describes the load-on-demand mechanism.
Predefined Readers introduces the predefined readers supplied with Rogue Wave Views Maps.
Map Projections explains how to use projections.
Map Data provides a list of suggested free sources for downloading map data.
