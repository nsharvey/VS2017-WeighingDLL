A grid (like a scale) can be displayed on top of (IlvDrawAbove) or underneath (IlvDrawBelow) the graphical representations of data in a chart. By default, a grid will be displayed on top of the graphical representations. However, the drawing order can be changed for a given grid by means of the setDrawOrder method.
If no palettes are defined to draw the major and/or the minor grid lines, the palette that will be used by default is the palette defined for the axis of the scale to which the grid is attached.
By default, a grid is drawn with only the major lines. To specify that the minor lines should also be drawn, you have to call the method drawMinorLines with IlvTrue as a parameter.
