Graph Layout
This manual provides information for developing applications that incorporate the Rogue Wave Views Graph Layout package. It describes the fundamentals that underlie the graph algorithms and shows how to create customized graph layouts.
This manual contains the following chapters:
Introducing the Graph Layout Package describes the Graph Layout package of Rogue Wave Views and its features.
Basic Concepts provides background information and basic concepts for using Graph Layout.
Getting Started with Graph Layout provides information to get started quickly using Graph Layout.
Layout Algorithms describes the layout algorithms provided with the Graph Layout package.
Using Advanced Features provides information on using a layout report, using layout event listeners, using a graph model, laying out a non-Views grapher, laying out a portion of a graph, laying out graphs with nonzoomable objects, and defining new types of layouts.
