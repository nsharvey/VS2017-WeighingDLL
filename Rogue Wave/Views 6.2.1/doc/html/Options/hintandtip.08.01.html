Hints and Tips for Using Data Access
This chapter contains some useful hints for using Data Access. The most common user requirements are described, along with tips on how to use the features of Data Access to satisfy these requirements. The chapter primarily discusses the way to set up your application using Rogue Wave® Views Studio, rather than using the Data Access API.
