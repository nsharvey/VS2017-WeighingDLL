General Elements
These elements apply to the SQL data source as a whole.
Element
Description
Name field
The SQL data source name that appears
under the SQL data source gadget in the work space
when Apply is clicked in the
SQL Data Source inspector panel.
Global checkbox
Default: Not checked. When checked, allows more than one user panel to use the current SQL data source.
Allow insert
checkbox
Default: Checked. When not checked, prevents the user from inserting a new row into the SQL data source tables, but does not prevent the user from editing existing rows.
