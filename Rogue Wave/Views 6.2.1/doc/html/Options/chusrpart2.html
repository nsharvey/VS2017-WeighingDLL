Part II 
Using the Charts Library
This part consists of the following chapters:
Introducing the Charts Library describes the main features of the Charts Library of Rogue Wave® Views.
Chart Basics provides background information and describes the basic concepts upon which the Charts Library is based.
Data Handling describes the data handling features of the Charts Library.
Chart Layout describes how the layout of the charts is computed.
Data Display describes how data are displayed using the chart displayers of the Charts Library.
Scales Display describes how scales are displayed using the scale displayers of the Charts Library.
Decorations Display describes how to add legends, grids, and annotations to your charts.
Interacting with Charts describes how to use the interactors of the Charts Library.
Using Charts to Display Real-Time Data tells you how to use charts to display real-time data.
