Installation of Rogue Wave Views Maps
In order to use the product, you need the following software:
Rogue Wave® Views Maps
Rogue Wave Views, Rogue Wave Views 2D Graphics Standard or Rogue Wave Views 2D Graphics Professional
If you want to use the Oracle Spatial reader of Rogue Wave Views Maps, you will also need to install Rogue Wave DB Link. The license of Rogue Wave Views Maps enables you to use Rogue Wave DB Link.
