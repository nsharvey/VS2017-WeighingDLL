Application Settings Directory
The directory where Rogue Wave Views 5.6 stores its settings has changed.
On Unix systems, this directory is now $HOME/.RogueWave. (This was changed from $HOME/.ILOG).
On Windows systems, this directory is called Rogue Wave in your user environment.
Warning: In order to keep your application settings when upgrading to Rogue Wave Views 5.6, you must move all content from the legacy directories to the new ones.
