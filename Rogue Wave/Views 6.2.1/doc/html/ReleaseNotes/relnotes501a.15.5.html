Execution Requirements
To ensure that every binary provided works properly, you need to set the environment variable ILVHOME to the directory where the product has been installed. Usually, you will set:
(csh/tcsh):
% setenv ILVHOME /usr/roguewave/viewsXX
(sh/bash):
$ ILVHOME=/usr/roguewave/viewsXX
$ export ILVHOME
(Windows 95/98/Me / Windows NT/2000/XP):
C:\> set ILVHOME=C:\Rogue Wave\VIEWSXX
Check with your system administrator the exact product installation directory.
