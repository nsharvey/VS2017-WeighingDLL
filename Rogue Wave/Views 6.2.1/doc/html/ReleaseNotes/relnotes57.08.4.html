Touch Screens
Rogue Wave Views now supports touch and multi-touch events.
Note: This feature is only available on Windows platforms for the version 5.7 release of Rogue Wave® Views.
New event types have been added to the enumeration IlvEvent::IlvEventType, allowing an application to handle touch events: IlvTouchDown, IlvTouchMoved, or IlvTouchUp.
The IlvDisplay class API has been extended to allow retrieving the touch points.
In this release, this support is only available on Windows platforms.
For more information see the corresponding Reference Manual entries.
