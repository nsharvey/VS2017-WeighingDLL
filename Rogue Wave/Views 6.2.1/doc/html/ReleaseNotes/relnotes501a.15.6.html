Library build information (Unix platforms only)
If a problem occurs while running your applications, you can retrieve the information on the platform that has been used to build the library, using the shell script called ilvversion.sh, located in the Rogue Wave Views root directory.
Go into the specific static subplatform subdirectory of 'lib', and type:
../../../ilvversion.sh
Two lines that have the following form will be printed out (these lines may be swapped):
IlvVersion: X.X
IlvBuild: OS - platform - compiler - display revision
The shell script just locates the strings IlvBuild andIlvVersion in the library file libxviews.a, and prints out the result.
If you have installed multiple platforms in the directory ILVHOME, you can check each of the platforms:
$ cd $ILVHOME
$ ./ilvversion.sh platform
where platform is the system identifier of the Rogue Wave Views library (that is, the <system>, one of hp32_11_3.30, rs6000_4.3_3.6, sparc_5_5.0...).
