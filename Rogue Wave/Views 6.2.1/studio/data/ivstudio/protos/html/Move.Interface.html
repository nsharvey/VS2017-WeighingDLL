Attributes Page: Moving Attributes

This panel allows you to define and edit the external properties of your group or prototype, for example, how you will access the group or prototype from your application or from other objects. You can define a set of attributes, give each of them a type and a default value.

Choose Edit->New Attribute or Ctrl+N to add an attribute.

You can also:

  • Import the interface of an other Prototype by choosing Edit->Delegate To Prototype
  • Order the attributes by selecting item Edit->Move Item Up or Edit->Move Item Down.
  • Directly edit the existing attributes by editing the appropriate row: name of the attribute, its type, its default value, and its flags: public, persistent, and notifying.
The views menu gives alternative presentations to the attributes of your prototype.
