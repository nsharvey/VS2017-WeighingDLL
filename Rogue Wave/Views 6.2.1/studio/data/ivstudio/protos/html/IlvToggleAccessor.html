Toggle

 When the value defined by the Toggle accessor (class IlvToggleAccessor) is modified, the value of another accessor is switched back and forth between true and false. The actual value assigned to the Toggle accessor is ignored.

Parameter

  • Boolean Accessor: Prototype value or node attribute that is switched back and forth when the accessor is evaluated. It must be a Boolean type (for example, the visibility attribute of the object).

Example

 The value toggle in the random prototype of the sources prototype library uses this accessor.
