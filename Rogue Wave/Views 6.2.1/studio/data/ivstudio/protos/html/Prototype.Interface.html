Delegate to Prototype

This function lets you add predefined attributes and behaviors coming from a chosen prototype.

After selecting Edit->Delegate To Prototype, choose in the list the prototype whose behavior you want to inherit. A new attribute is created, and the inherited attributes are shown in the table grayed-out.

You cannot edit directly these inherited attributes, but you can reference them in other attributes.

Some inherited attributes may reference other attributes or graphic nodes. Therefore, not every behavior can be inherited into any prototype.
