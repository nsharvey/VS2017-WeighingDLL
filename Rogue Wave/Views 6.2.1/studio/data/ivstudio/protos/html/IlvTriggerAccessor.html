Assign

The Assign accessor (class IlvTriggerAccessor) is used to assign a value to a node attribute or another value. When the property handled by the accessor is set, the target property specified by the target parameter is set to the specified value.

The Assign accessor is useful when a property depends on another property. It allows you to recompute the target property when the property on which it depends is modified.

Parameters

  • Input: Value that is copied to the output.

  • Accessor: Prototype value or node attribute to which input is copied. The type of the accessor is undetermined and irrelevant.

Example

 The lcd2 prototype of the lcd library uses the Assign accessor.
