Invert

 The Invert accessor (class IlvInvertAccessor) inverts the color of an element of your prototype periodically. When the property is set to IlvTrue, the color inversion starts. When the property is set to IlvFalse, the color inversion stops.

While the colors are designated as the foreground and background colors, any colors defined by the prototype or one of its nodes can be used.

Parameters

  • Fg Col. Accessor: Node attribute or prototype value that contains the foreground color.

  • Bg Col. Accessor: Node attribute or prototype value that contains the background color.

  • Period (ms): The interval (in milliseconds) between two inversions of the object colors. It can be Float or Integer.

  • Type: Boolean (whether the accessors are exchanging their values or not)

Example

 An example is presented in the pump prototype of the samples library. When invert is set to true, the values of rotorColor and invertedColor are exchanged periodically. The period is defined by the invert value. Note that the invertPeriod value has an assign invert <- invert accessor defined. This forces the accessors to be reevaluated and the internal timer to update its period whenever the period is changed.
