Watch

The Watch accessor (class IlvLoopbackAccessor) defines an input value that is set automatically when a given output value is generated. It is functionally equivalent to an internal value subscription.

This accessor class is often used with the Callback accessor to change a value of the prototype when a callback is triggered. The Connect accessor connects the trigger value generated by the callback to the input value that must be changed.

Parameters

  • Notifying Attribute: Attribute that triggers the accessor value. This node must be one of the values that have a Notify, Callback, or Event behavior (that is, nodes that can push a value).
