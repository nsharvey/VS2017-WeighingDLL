Increment

The Increment accessor (class IlvCounterAccessor) is used to increment a specified value handled by a group. Each time the property handled by this accessor is set, another value, called a counter, is incremented by one until a specified maximum value is reached. When this value is reached, the counter is reset to zero.

Parameters

  • Maximum: Maximum value. The value to increase is reset to 0 if it is equal to the maximum value.

  • Accessor: Prototype value or node attribute to increment.

  • The type of the accessor is undetermined and irrelevant.

Example

 A three-state button can be implemented by using an Increment accessor linked to a MultiRep accessor.
