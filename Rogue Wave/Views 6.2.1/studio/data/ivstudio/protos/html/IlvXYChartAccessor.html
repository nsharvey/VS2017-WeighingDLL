Chart

 The Chart accessor (class IlvXYChartAccessor) defines two values that correspond to the points that are added to IlvChart data. Changing these two values in a single call to changeValues on the prototype (or one of its instances) adds a point at the end of the chart.

The prototype holding this accessor must contain an IlvChart object and two IlvRectangularScale objects (vertical and horizontal).

If only the Y value is changed (and not the X value), the X value is assumed to be the previous X value incremented by 1.

You can specify any name and type for this accessor since the actual values it defines are the values specified as the X Value name and Y Value name parameters.

Parameters

  • Chart Node: Name of the node containing the IlvChart.

  • X Scale Node: Name of the node containing the horizontal scale. Must be an IlvRectangularScale object.

  • Y Scale Node: Name of the node containing the vertical scale. Must be an IlvRectangularScale object.

  • X Value Name: Value name of the X coordinate of the point to add.

  • Y Value Name: Value name of the Y coordinate of the point to add.

  • The type of this accessor is undefined and irrelevant because its value is not used.
