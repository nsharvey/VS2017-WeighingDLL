Min/Max

The Min/Max accessor (class IlvMinMaxAccesor) is similar to the Condition accessor but handles common cases when a value must be tested against a minimum and a maximum threshold. When the value of this accessor is changed, another accessor is set depending on whether the value is less than the minimum, between the minimum and the maximum, or greater than the maximum. In addition, an exception condition can be specified: if the exception condition is true, no value is changed.

Parameters

  • Minimum: Defines the minimum value.

  • Maximum: Defines the maximum value.

  • Except if: If this value is true, the value is ignored and the output value or attribute is not set. The expression must result in a Boolean value.

  • Accessor: Prototype value or node attribute that is set to one of the following three values:

  • If x < min: Value to which the prototype or node attribute is set if the value is less than the minimum.

  • If min < x < max: Value to which the prototype or node attribute is set if the value is between the minimum and the maximum.

  • If x > max: Value to which the prototype or node attribute is set if the value is greater than the maximum.
