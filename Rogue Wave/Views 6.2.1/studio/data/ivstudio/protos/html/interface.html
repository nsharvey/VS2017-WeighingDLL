Defining the Attributes

First, you should define and edit the external attributes (also called properties) of your group. That is to say, how you will access it from your application or from other objects. This page allows you to define a set of attributes, giving each of them a type and a default value.

Choose Edit->New Attribute or Ctrl+N to add an attribute. A new row "Unnamed" appears in the table. Specify the name, type (press F2 if using only the keyboard), default value, and the public, persistent, and notifying flags of the new attribute in the appropriate columns.

You can also:

  • Import the interface of an other Prototype with Edit->Delegate To Prototype
  • Order the attributes by selecting an item and Edit->Move Item Up or Edit->Move Item Down.
  • Delete an attribute: select it, and then Edit->Delete.
  • Start drawing your prototype in the prototype buffer window.

The View menu provides alternative presentations to the attributes of your group or prototype.
