Notify

 The Notify accessor (class IlvOutputAccessor) turns changeValue calls on the value to which it is attached to pushValue calls. Values that are connected to the attached value will execute all their accessors.

This accessor triggers behaviors of other values that depend on the trigger value. For example, you can make a change in the Threshold value to also reevaluate the Temperature value. This can be done by attaching a Trigger accessor to the Threshold value.
