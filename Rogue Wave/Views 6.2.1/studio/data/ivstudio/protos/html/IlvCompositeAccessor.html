Multiple

The Multiple accessor (class IlvCompositeAccessor) defines a value that is copied to or from several other values or attributes. It can be used, for example, to change the colors of two graphic nodes using a single public value of the prototype.

Parameters

  • Accessor: Prototype value or node attribute to or from which the value is copied.

  • Accessor: Prototype value or node attribute to or from which the value is copied.

  • All accessors must have compatible types and the type of the value must be a supertype of all of the output types. For instance, if there are integer and float outputs, the value must be a float type, which is more general than integer.
