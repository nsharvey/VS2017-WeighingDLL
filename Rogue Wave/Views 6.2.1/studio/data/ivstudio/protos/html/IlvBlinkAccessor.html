Blink

The Blink accessor (class IlvBlinkAccessor)makes an object of your prototype blink (that is, it causes the object to appear and disappear periodically). When the property is set to IlvTrue, the object starts blinking. When the property is set to IlvFalse, the blinking stops.

Parameters

  • Boolean Accessor: Object attribute that controls the object visibility; for example, light, visibility.

  • Period (ms): The interval (in milliseconds) between two "blinks". It can be Float or Integer.

  • The type of this value is Boolean.
