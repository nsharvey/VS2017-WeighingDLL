ExportTrigger

The ExportTrigger accessor (class IlvExportOutputAccessor) is equivalent to a combination of an Export accessor and a Trigger accessor. This accessor is attached to an attribute of a node of the prototype. When the attribute is changed (either by the application or some other behavior of the prototype), its value is "pushed" to whatever values can be connected to the value attached to this accessor, using a Connect accessor.

Parameters

  • Accessor: The node attribute the accessor references (see the Export accessor). When this attribute is changed, a trigger (or pushValue) of the value is done.
