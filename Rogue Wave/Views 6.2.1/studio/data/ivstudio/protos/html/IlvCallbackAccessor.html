Callback

This accessor (class IlvCallbackAccessor) attaches a trigger that is set when the given callback is called from a user action on the specified graphic node. For a callback to be called, the node must be a gadget or an interactor must have been attached to the graphic node that is to trigger callbacks.

Parameters

  • Graphic Node: The name of the graphic node whose callback causes the trigger when it is fired.

  • Callback Name: The name of the callback.

  • Input: The value sent when the callback is triggered.
