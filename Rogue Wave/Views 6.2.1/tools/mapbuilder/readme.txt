                              Rogue Wave Views Maps 6.2

                                The Map Builder

The Rogue Wave Views Maps Builder is an editor for creating maps that can be
loaded into applications.

The Map Builder is provided with its source code. This source code is
located in the directory <installdir>/tools/mapbuilder.

To compile the mapbuilder, you will need the Rogue Wave Views Gadget and
Rogue Wave Views Managed packages and the Rogue Wave DB Link package.

To run the mapbuilder with Oracle Spatial enabled, you must install the
Oracle Client on your machine, and of course have access to an Oracle database
server.

See the "Getting started" section of the user manual for more informations
on the capabilities of the Map Builder.
