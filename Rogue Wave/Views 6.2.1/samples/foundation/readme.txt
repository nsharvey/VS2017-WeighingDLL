
                             Rogue Wave Views 6.2
                             Foundations Package

                                   Samples

  This directory contains samples for Rogue Wave Views Foundation.


Directory: basic
================
  Demonstrations of basic functionalities of Rogue Wave Views, such as
  manipulating graphic objects and interactors.


Sample: 'ilvread'
-----------------
  Loading and displaying a Rogue Wave Views data file.


Sample: 'inter'
---------------
  Displaying and interacting with graphic objects.


Sample: 'hull'
--------------
  Using IlPool in a recursive algorithm.

Directory: bitmaps
==================
  Demonstrations of the Rogue Wave Views bitmap readers. These samples works
  only in true color mode.


Sample: 'readbm'
----------------
  Loads and displays supported bitmap images.


Sample: 'createbm'
------------------
  Shows you how to use bitmap data.


Sample: 'quantize'
------------------
  Shows you how to use the color quantizer.


Sample: 'animbm'
----------------
  Shows you how to use bitmap data operators.


Sample: 'jpgcomp'
-----------------
  Shows you how to use the JPEG streamer.


Sample: 'animate'
-----------------
  Shows you how to use the IlvBitmapAnimationHandler class.


Sample: 'alpha'
---------------
  Shows you how to use alpha blending.


Sample: 'transition'
--------------------
  Shows you how to use the IlvBitmapTransitionHandler class.


Sample: 'contrast'
------------------
  Shows you how to use the IlvIndexedBitmapData class.

Directory: bmpfilters
=====================
  Demonstrations of the Rogue Wave Views bitmap filters.


Sample: 'displace'
------------------
  Shows how to distort an image using turbulence.


Sample: 'emboss'
----------------
  Shows how to achieve an embossing effect.


Sample: 'filtgraph'
-------------------
  Shows how to use the the IlvFilteredGraphic class.


Sample: 'gold'
--------------
  Shows how to use bitmap filters to create a gold effect.


Sample: 'relief'
----------------
  Shows how to use bitmap filters to create a relief effect.

Directory: gpath
================
  How the IlvGraphicPath object can be used.


Sample: 'gpath'
---------------
  Using the capabilities of the IlvGraphicPath object: disconnected polylines,
  polygons with holes and drawing along a path.

Directory: multitouch
=====================
  Demonstrates the support for Multi-Touch screens in Rogue Wave Views.


Sample: 'multitouch'
--------------------
  Demonstrates how to use Views multitouch events

Directory: printing
===================
  How to use the printing framework.


Sample: 'headfoot'
------------------
  Header and footer usage with IlvPrintableLayout.


Sample: 'photops'
-----------------
  An example of how to print using Postscript printing.


Sample: 'photpsdlg'
-------------------
  An example of how to print using an IlvPSPrinter and an
  IlvPostscripPrinterDialog instance.


Sample: 'photpspr'
------------------
  An example of how to print using Postscript printing with a printing preview
  displayer.


Sample: 'photowin'
------------------
  An example of how to print using a Windows printer.


Sample: 'phowindc'
------------------
  An example of how to print using IlvWindowsPrinter and an
  IlvWindowsPrinterDCFactory subclass.


Sample: 'phowinpr'
------------------
  An example of how to print using IlvWindowsPrinter printing with a printing
  preview displayer.

Directory: table
================
  How the IlvTable object can be extended.


Sample: 'prime'
---------------
  An example of how to extend an IlvTable that can be used in a specialized
  graphic object.

Directory: windows
==================
  Windows specific samples.


Sample: 'ilvwin'
----------------
  Integrate Windows components into a Rogue Wave Views application.


Sample: 'winilv'
----------------
  Integrate Rogue Wave Views components into a Windows application.


Sample: 'metafile'
------------------
  Convert Rogue Wave Views data files to a Windows metafile.


Sample: 'ilvmdi'
----------------
  Use of the IlvMDIChild property on Rogue Wave Views views.


Sample: 'gdiplus'
-----------------
  This sample shows the GDI+ features used in Rogue Wave Views.

Directory: xlib
===============
  X11 specific samples.


Sample: 'usewin'
----------------
  This sample demonstrates how to take control over any existing X Window.


Sample: 'ilvmotif'
------------------
  This sample demonstrates how to take control over any existing Motif gadget.


Sample: 'mpanim'
----------------
  This sample demonstrates how to use the Rogue Wave Views capability of
  accessing X Windows bitplanes. Note that this sample fully makes sense only
  on X visual that are PseudoColor.

Directory: motif
================
  Motif specific samples.


Sample: 'ilvmotif'
------------------
  This sample shows you how to create a Rogue Wave Views application on top of
  a Motif application.


Sample: 'dragdrop'
------------------
  Demonstration of Motif Drag-and-Drop from and to Rogue Wave Views.


Sample: 'multidd'
-----------------
  Another demonstration of Motif drag-and-drop between two separated Rogue Wave
  Views applications.


Sample: 'scrollhk'
------------------
  This sample shows how to connect an IlvManagerViewHook to a Motif
  ScrolledWindow gadget.

Directory: i18n
===============
  Demonstrations of internationalization functionalities of Rogue Wave Views.


Sample: 'changelang'
--------------------
  This sample demonstrates how to use localized message databases and change
  the language of the UI.


Sample: 'controlinput'
----------------------
  This sample demonstrates how to control localized input in input fields.
