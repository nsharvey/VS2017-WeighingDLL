utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'text'
--------------
  Example of a simple text editor.

  This program lets you edit text files and change the font parameters of the
  displayed text. It shows you how to manage gadgets inserted in a toolbar (for
  example, a combobox) using the event mechanism of the Application Framework
  library.

  Classes involved:
    - IlvDvTextDocument
    - IlvDvTextView

  Source files:
    - include/TextDocument.h
    - include/TextView.h
    - src/TextDocument.cpp
    - src/TextView.cpp
