Sample: polar

Displaying a Polar Chart

Creation of a simple polar chart with several displayers.

This sample shows how to create a chart using a polar projector. To do so, it uses the IlvPolarChart class.

Note the two different scale types: rectangular for the rho coordinate that is displayed radially, and circular for the theta coordinate.

Two callback functions are used as input data and are rendered with two displayers:

  • A polygon representation representing the function y = abs(2*sin(2x)).
  • A bar representation representing the function y = abs(sin(3x)).

Classes involved:

  • IlvPolarChart
  • IlvCallbackChartFunction
  • IlvPolygonChartDisplayer
  • IlvBarChartDisplayer

Source files:

  • ../include/samputil.h
  • ../src/samputil.cpp
  • src/polar.cpp
