The Data Access samples use two database tables. You can use your own database system tools to create the two tables, using the SQL statement below as a model. These statements are based on the SQL2 standard.
Alternatively, you can use the 'SQL Schema Editor' option in 'dbstudio' program with file 'samples/dataccess/samples.ilx'.

Data Access Sample Database Schema

Table I_EMP

   CREATE TABLE I_EMP (
	   EMPNO   INTEGER         NOT NULL,
	   NAME    VARCHAR(25)     NOT NULL,
	   STATUS  INTEGER         NOT NULL,
	   DEPTNO  INTEGER         NOT NULL,
	   SALARY  REAL            NULL,
	   PRIMARY KEY (EMPNO))

Table I_DEPT

   CREATE TABLE I_DEPT (
	   DEPTNO  INTEGER         NOT NULL,
	   NAME    VARCHAR(25)     NOT NULL,
	   MANAGER VARCHAR(25)     NOT NULL,
	   PRIMARY KEY (DEPTNO))
