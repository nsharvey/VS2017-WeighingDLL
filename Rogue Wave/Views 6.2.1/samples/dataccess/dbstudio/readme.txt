To compile the programs, go to the directory <system>, and run the 'make'
utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'dbstudio'
------------------
  This sample defines an Rogue Wave Views Studio with static Data Access
  libraries. The Manager package is needed for this sample.

  This C++ sample:

    * Links the Rogue Wave Views Studio with the Data Access plugin

  Source files:
    - src/main.cpp
    - src/initrt.cpp
