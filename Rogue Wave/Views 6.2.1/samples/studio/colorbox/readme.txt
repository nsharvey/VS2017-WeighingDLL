
                             Rogue Wave Views 6.2
                               Gadgets Package

                              Samples 'colorbox'


To compile the programs, go to the directory <system>, and run the 'make'
utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'studio'
----------------
  Shows an extension making an Inspector for Rogue Wave Views Studio.

  The colorbox program is an extended version of Rogue Wave Views Studio that
  implements an inspector for a customized gadget.
