Sample: studio

Studio Extension Sample

Shows a complete extension for Rogue Wave Views Studio.

The mystudio program is an extended version of Rogue Wave Views Studio that has new kinds of buffers, objects, commands, and so on.

Try the new commands: AddMyClass, MyNewBuffer and MyShowPanel.
