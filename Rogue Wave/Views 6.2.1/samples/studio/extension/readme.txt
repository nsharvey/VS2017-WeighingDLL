
                             Rogue Wave Views 6.2
                               Gadgets Package

                             Samples 'extension'


To compile the programs, go to the directory <system>, and run the 'make'
utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'studio'
----------------
  Shows a complete extension for Rogue Wave Views Studio.

  The mystudio program is an extended version of Rogue Wave Views Studio that
  has new kinds of buffers, objects, commands, and so on.

  Try the new commands: AddMyClass, MyNewBuffer and MyShowPanel.
