Sample: inspect

Rogue Wave Views Gadgets Generic Inspector

This sample shows how to implement a small generic inspector using the IlvHierarchicalSheet class.

You can load Rogue Wave Views files and inspect their content in the hierarchical sheet. The items of the first level are the names of the loaded files. When you expand them, the sheet displays all the objects contained in the corresponding file. Then you can inspect each object property, and modify it, the changes are automatically applied.

Classes involved:

  • IlvHierarchicalSheet
  • IlvTreeGadgetItem
  • IlvLabelMatrixItem
  • IlvGadgetItem
  • IlvButton
  • IlvValue
  • IlvFileBrowser
  • IlvPathName
  • IlvGadgetContainer
  • IlvGadgetItemHolder
  • IlvGraphicHolder
  • IlvPalette
