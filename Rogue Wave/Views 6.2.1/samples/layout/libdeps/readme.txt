
                             Rogue Wave Views 6.2
                             Graph Layout Package

                              Samples 'libdeps'


To compile the programs, go to the directory <system>, and run the 'make'
utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'libdeps'
-----------------
  Performs the Hierarchical layout on the Rogue Wave Views libraries graph.

  The Gadgets package must be installed to be able to build this sample. Use
  the Space or Ctrl+L accelerators to perform the Hierarchical layout.
  Use the Ctrl+P accelerator to create a Postscript dump of the current graph.
  Use the Ctrl+F or Ctrl+S accelerator to search for a library.

  Classes involved:
    - IlvHierarchicalLayout

  Source files:
    - data/dialog.ilv
    - data/libdeps.xml
    - include/dialog.h
    - include/objsel.h
    - src/dialog.cpp
    - src/libdeps.cpp
    - src/objsel.cpp
