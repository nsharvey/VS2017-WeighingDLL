
                             Rogue Wave Views 6.2
                             Graph Layout Package

                             Samples 'inspector'


To compile the programs, go to the directory <system>, and run the 'make'
utility provided on your system.
Some ports are provided with compiler-specific project files.
Run the samples from the <system> directory.

Sample: 'inspector'
-------------------
  The Gadgets package must be installed to be able to build this sample.
  Shows how to create layout instances and allows you to perform different
  layouts on the same graph.


  Classes involved:
    - IlvBusLayout
    - IlvHierarchicalLayout
    - IlvOrthogonalLinkLayout
    - IlvRandomLayout
    - IlvTreeLayout

  Source files:
    - include/panel.h
    - src/inspector.cpp
    - src/panel.cpp
    - data/bus.ilv
    - data/hierarchical.ilv
    - data/mainpan.ilv
    - data/orthogonal.ilv
    - data/random.ilv
    - data/tree.ilv
